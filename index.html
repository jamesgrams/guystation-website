GuyStation

About

GuyStation is an Emulator Hub for Ubuntu. It is a small frontend for emulators designed to be easily installable and accessible. In addition to having several emulators, GuyStation's major highlights include the ability to stream games to and access the menu from any device on the same network, the ability create seperate save files for the same game (designed for Pok√©mon :)), a media player that lets you play music or videos on any device on the same network, and an internet browser that can be controlled remotely.

Features

  • Play Classic Games
    Support for Game Boy, Game Boy Color, Game Boy Advance, NES, Nintendo DS, Nintendo 3DS, Nintendo 64, Nintendo GameCube, Wii, PC, PlayStation 1, PlayStation 2, PSP, and SNES
  • Remote Play
    Play from any device with a web browser in your house. GuyStation will stream to your device, and you can control GuyStation remotely with a keyboard, controller, or phone's touch screen.
  • Remote Control
    GuyStation's interface is a webpage that is made accessible on your network. This means you can add games, add saves, switch saves, and more from any device in your house.
  • Multi-Save
    Easily create and switch between save files for any game. This allows for multiple adventures on games that limit you to one. Your screenshots are associated with your save file.
  • Internet Browser
    Access any website on your TV and control the browser with any device you like. Additional features allow you to play other consoles in-browser through HDMI input, given you have the correct hardware. Advanced users can create browser scripts to perform functions such as automatically playing popular videos on Disney+!
  • Media Player
    Play videos and music and stream them to any device in your house. Create playlists from your files for maximum fun!
  • Easy Controller Configuration
    Set the controls for all systems at once using GuyStation's built-in controller configuration. You can save configurations as profiles and set devices to load profiles automatically when you play a game from them. Want to use gamepad controls on the TV and keyboard controls when playing remotely on your computer? No Problem!
  • Direct ROM Download
    In addition to uploading games to GuyStation from any device, if you have a link to a ROM or zip file containing a ROM, you can simply enter that link, and GuyStation will download it and extract it ready for you to play. This works for YouTube videos as well for the media player.
  • Picture-in-Picture Mode
    You can watch videos in Picture-in-Picture while you play and easily switch between full-screen and picture-in-picture mode. You can play videos from any URL with a video on it or media you have downloaded to GuyStation.
  • IGDB Game Info
    Following the necessary setup, GuyStation will display cover art, release dates, and descriptions for every game it can find in the IGDB.
  • Seamless Screenshot Integration
    You can use the controller configuration to set one keyboard key to take screenshots on all emulators. Your screenshots will be accessible through GuyStation's web interface.
  • Messages
    Messages can be sent between GuyStation users on the same network.

Get Started

  1. Install Ubuntu 18.04 on the computer that you want to use.
  2. Download and unzip the GuyStation installer on the computer you set up in step one.
  3. Double-click the unzipped file named `runme.desktop` (AKA `GuyStation Installer`) to run it.
  4. Wait for the script to finish, reboot the computer, and GuyStation will pop up!
    • There are several optional quality-of-life changes you can make to the computer. For example, preventing the screen from going to sleep. These recommendations are listed here.
    • You will have to make some tweaks to get every emulator running optimally. Those tweaks are listed here.
See the GuyStation Readme for full documentation.
